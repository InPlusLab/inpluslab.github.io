TPL Dataset

Third-Party Library Dataset

Description

We crawled 10148 apps from Android markets and unpack them one by one. Third-party libraries are searched by traversaling every folder in the app. We utilize the library list carried out by Viennot et al. with about 352 libraries. Based on this list, we drop those apps without adopting any of these libraries leaving 8695 apps. In the dataset, each row of the app-lib matrix represents the libraries adopted by the current app, where the value of 1 indicates that a library is adopted by an app and 0 vice versa.
