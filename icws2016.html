Wandoujia Dataset

Description

This dataset is from Wandoujia, a leading Android application marketplace in China. The dataset is the log of users' download and uninstall records from May 1, 2014 to July 31, 2014, which includes 688198 log records of 5057 users and 4496 applications.


The Wandoujia dataset includes:

  1. Download record. Format: | status | userID | appID | Unix timestamp |
    2. status userID App ID timestamp
    success 3 com.tencent.mobileqq 1398960000
    failure 3254 com.tencent.mm 1400083200
    success 0 com.immomo.momo 1402761600
  2. Uninstall record. Format: | status | userID | appID | Unix timestamp |
    3. status userID App ID timestamp
    success 1167 com.eg.android.AlipayGphone 1405353600
    failure 4120 com.taobao.taobao 1404144000
    success 5 com.immomo.momo 1401552000
